Str() - Função. Recebe um número e retorna uma string com esse número formatado.

QOut() - Procedimento. Executa a ação de exibir algo na tela.

Date() - Função. Retorna a data atual formatada com o formato de data, não string.

Len() - Função. Retorna um número indicando o tamanho da string ou do array.

Upper() - Função. Retorna uma string convertida para letras maiúsculas.

ClearScreen() - Procedimento. Apenas executa a ação de limpar a tela.
